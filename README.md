## 游戏场景生成图

很多互联网产品在首页呈现用户场景时用了类似下图，每个图都需要前端来实现，即使很多素材可复用，但依旧会花费很多工时。

创建一个游戏，包含很多风格上类似下图的素材，可以自己搭建场景，并让其中的人物可以活动。用户构造好场景后，可以捕捉当前画面，直接作为UI图。
